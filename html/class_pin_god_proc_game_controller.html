Detailed Description

A PinGod P-ROC Game controller. This overrides methods from the P-ROC BaseGameController.

This GameController is built using an IFakeProcDevice, but that could be a real* IProcDevice board by switching off simulation. *The project and Godot would have to be x86 if real

Add modes to BallStarting and remove them from BallEnded (this is if the mode starts and ends here of course)

GameStarted adds the ScoreDisplay. (Attract is removed from switch handler in that mode).

GameEnded removes the ScoreDisplay and adds the attract (which is newed up)
