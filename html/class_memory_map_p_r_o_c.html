override void MemoryMapPROC.ReadStates ( )
protectedvirtual

Reads the buffer size TOTAL_SWITCH switch states from the memory map buffer at position 0. Acts on any new switch events found. The switch state gets converted to a Godot InputEventAction with the name sw{Num} and fed into the gameIt is overidden if a VpCommand found. Switch 0 changed will process game states, switch zero used with GameSyncState.
