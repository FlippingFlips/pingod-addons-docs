Detailed Description

Resources, graphic packs. Set to AutoLoad with a Resources.tsn in autoload directory.

Select Packed scenes you wish to get to in the game. You can get these by full path res:// and removing the extension .GetBaseName()

Then these scenes can be instantiated.

A ResourcesLoaded signal will be sent when all resources have finished loading with the amount of resources loaded.

It works by using the _ProcessThread and loading scenes in background. (see _Process)

Processing is disabled when all packed have been loaded
