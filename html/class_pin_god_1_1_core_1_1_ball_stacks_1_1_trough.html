Detailed Description

Simple simulation of a Pinball ball trough. Handles trough switches, ball saves

Loaded as singleton on launch. Visual pinball uses the bsTrough in example. Emits PinGodGame signals for ball saves, ball end

Trough Exports for Godot editor
