Detailed Description

This window relies on a child having a Control node which contains playfield image and buttons.

see "res://addons/pingod-editor-plugin/machine-plugins/playfield/playfield_control.tscn"

This child control is made just with godot script for a signal when the buttons are pushed.

What you would do for your game is duplicate this scene and make your own from it, replacing the image and button names to your switch names.

You would want to duplicate the scene for this script too and set the switch window in the MachineNode scene.

Defaults to 400x900 size with a playfield at 400x908
