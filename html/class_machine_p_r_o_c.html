Detailed Description

This script is loaded with the res://autoload/Machine.tscn which inherits a PinGod-MachineNode.

Singleton can be retrieved from the /root/ with GetNode<MachinePROC>("/root/Machine").

EnterTree initializes a database, gets machine config
